Mobile Dev Toolkit

More Platforms. More Problems.

Sadly, we don't all have multiple native development teams.

The Downsides of Native

  • Proficiency in each platform required
  • Entirely separate code bases
  • Time consuming & expensive development
  • Diminishing returns

    • Is there an alternative?

Hybrid Apps!

  • HTML5 that acts like native
  • Web wrapped in native layer
  • Direct access to native APIs
  • Cordova/Phonegap
  • Familiar web dev environment

"I tried PhoneGap once and it was slooooooowwwwww"


Good News:

Mobile devices have rapidly improved!

Hybrid app performance is finally catching up.

So What Do We Need?

  • An Open Source Framework
  • Help bridging the gap between web and native
  • Rich, native-style UI components and interactions
  • UI APIs, not just jQuery widgets
  • A dedicated community

The following slides are a very basic tutorial for manipulating an Ionic seed project. Project would be started as follows:


            $ npm install -g cordova ionic

$ ionic platform add ios

$ ionic build ios
$ ionic emulate ios

$ ionic serve

Wait, what is happening??!!

very basic overview..

'build' compiles app from www folder (assets) and config.xml (preferences)

HTML/CSS/Javascript is wrapped into a webview

index.html loads Ionic CSS, Ionic JS, cordova.js, and our AngularJS app, ng-app="starter"

ion-nav-bar and ion-nav-view are updated by js and templates

Ionic Land

very basic overview of AngularJS in this example

app.js - routes setup & "onload" function

controller.js - relays data and actions within scope to view

services.js - example factory for returning static data

templates - individual screens html

.controller('FriendsCtrl', function($rootScope, $scope, $http, Friends) {
    $http.get('http://ajax.googleapis.com/ajax/services/feed/load?v=1.0&num=100&q=http://feeds.feedburner.com/asyncjs').then(function(resp) {
       console.log('Success');
       $scope.friends = resp.data.responseData.feed.entries;
       //rootScope is not the best option, but works for demo purposes

6. Add username prompt to Account tab

//Replace in controllers.js


.controller('AccountCtrl', function($scope, $ionicPopup) {

  $ionicPopup.prompt({
    title: 'Fancy Login',
    subtitle: 'Enter your username',
    inputType: 'text',
    inputPlaceholder: 'Username'
    }).then(function(res) {
      if (res){
        $scope.showAlert(res);
      }
  });

  $scope.showAlert = function(username) {
    var alertPopup = $ionicPopup.alert({
      title: 'Welcome!',
      content: username
    });
      alertPopup.then(function(res) {
      console.log('Alert closed');
    })
  }
     
});
